# 露天矿关键生产指标分析看板 - 功能使用说明文档

## 一、系统概述

露天矿关键生产指标分析看板是一套集成化的生产数据分析工具，通过直观的数据可视化展示，
帮助矿山管理者全面掌握生产运营状况，及时发现问题并做出科学决策。

系统包含五大核心模块：
1. **产量分析** - 多维度采剥总量分析与计划完成率监控
2. **设备效率** - 电铲与矿车效率分析及低效设备预警
3. **损失贫化** - 矿石损失率、贫化率及品位对比分析
4. **爆破效果** - 炸药单耗与块度合格率的关系研究
5. **成本分析** - 四项成本构成拆解与对比分析

---

## 二、快速入门

### 2.1 启动系统
1. 按照部署说明完成环境配置
2. 执行命令启动应用：
   ```bash
   streamlit run app.py
   ```
3. 在浏览器中打开显示的地址（默认 http://localhost:8501）

### 2.2 界面导航
系统采用左侧导航 + 右侧内容区的经典布局：

- **左侧导航栏**：选择分析模块、查看使用说明、了解数据信息
- **右侧内容区**：展示选中模块的数据分析图表和交互控件
- **顶部工具栏**：Streamlit 提供的通用操作（重新运行、设置等）

### 2.3 数据准备
- 首次启动系统会自动检测数据文件并生成模拟数据
- 也可手动执行 `python generate_data.py` 生成或重置数据
- 所有数据文件保存在 `data/` 目录下，可替换为真实生产数据

## 三、功能模块详解

### 3.1 系统概览

**功能定位**：全局数据总览，快速把握生产全貌

**核心内容**：
1. **关键指标 KPI 卡片**
   - 近6月采剥总量：反映近期整体生产规模
   - 电铲平均装车效率：设备装载作业效率参考值
   - 平均损失率/贫化率：资源利用效率核心指标
   - 爆破最佳区间占比：爆破参数优化程度
   - 近6月平均单位成本：生产成本总体水平

2. **功能模块介绍卡片**
   展示五大分析模块的功能概要，引导用户进入具体分析。

**操作方式**：
- 直接查看，无需额外操作
- 点击左侧导航栏进入具体模块进行深入分析

### 3.2 产量分析

**功能定位**：全面监控产量计划执行情况，发现产量波动异常

**筛选控件**：
| 控件名称 | 功能说明 | 可选值 |
|---------|---------|-------|
| 时间维度 | 选择数据聚合粒度 | 按日 / 按月 / 按年 |
| 指标类型 | 选择分析的产量指标 | 采剥总量 / 矿石量 / 岩石量 |
| 时间范围选择器 | 限定分析的时间区间 | 根据所选维度动态变化 |

**分析图表**：

1. **KPI 指标卡（4个）**
   - 实际产量：选定时间范围内的累计实际产量
   - 计划产量：对应期间的计划产量
   - 完成率：实际/计划 × 100%，附带与目标的偏差
   - 平均周期产量：每单位时间（日/月/年）的平均产量

2. **产量趋势与计划对比图**
   - 蓝色柱状图：实际产量
   - 红色折线图：计划产量
   - 交互功能：悬停查看具体数值，点击图例隐藏/显示数据系列

3. **计划完成率趋势图**
   - 绿色折线：完成率变化趋势
   - 红色虚线：100% 目标线
   - 可直观看到哪些时段未完成计划

4. **数据明细表**
   - 展示各周期的实际值、计划值和完成率
   - 支持滚动浏览和全表查看

5. **数据下钻功能（年→月→日）**
   - 在"按年"视图下：选择年份查看各月详情
   - 在"按月"视图下：选择月份查看各日详情
   - 实现层层递进的数据分析

**使用场景**：
- 每日早会：快速了解昨日产量完成情况
- 月度总结：分析本月产量波动原因
- 年度考核：评估全年产量目标完成情况

### 3.3 设备效率

**功能定位**：监控电铲和矿车的运行效率，识别需要检修的低效设备

**筛选控件**：
| 控件名称 | 功能说明 | 可选值 |
|---------|---------|-------|
| 统计周期 | 选择效率计算的时间范围 | 任意日期区间 |
| 设备类型 | 选择分析的设备类别 | 全部设备 / 电铲 / 矿车 |
| 排序方式 | 设备列表排序规则 | 效率从低到高（优先关注）/ 效率从高到低 |

**分析图表**：

1. **总体 KPI（4个）**
   - 平均电铲装车效率（秒/车）：数值越低越好
   - 平均矿车运输效率（趟/班）：数值越高越好
   - 累计装车数
   - 累计运输量

2. **电铲效率分析**
   - **柱状图**：各电铲平均装车效率对比
     - 蓝色：效率正常
     - 黄色：效率偏低
     - 红色：效率过低（需重点关注）
     - 灰色虚线：全体平均值
   - **单设备趋势图**：选择特定电铲查看其效率随时间的变化
   - **明细表**：各电铲的完整统计数据

3. **矿车效率分析**
   - **柱状图**：各矿车平均运输效率对比（颜色含义同上电铲）
   - **班次对比图**：早班/中班/晚班的效率差异分析
   - **明细表**：各矿车的完整统计数据

4. **低效设备检修建议（TOP 5）**
   - 自动筛选效率最低的前5台电铲和前5台矿车
   - 展示设备编号、效率指标、累计工作量、运行时长
   - 根据运行时长自动给出检修建议：
     - 运行时长 > 2000小时：建议全面检修
     - 运行时长 1500-2000小时：建议中期维护
     - 运行时长 < 1500小时：建议检查操作和易损件

**使用场景**：
- 设备调度：合理安排高效设备承担重点任务
- 检修计划：根据效率排名制定设备维护计划
- 班组考核：对比各班次设备运行效率差异

### 3.4 损失贫化

**功能定位**：监控采矿过程中的资源损失和质量贫化情况

**筛选控件**：
| 控件名称 | 功能说明 | 可选值 |
|---------|---------|-------|
| 采矿区域 | 选择分析的作业区域 | 全部区域 / A区 / B区 / C区 / D区 |
| 时间范围 | 选择分析的时间区间 | 任意日期区间 |
| 仅显示异常数据 | 过滤开关 | 勾选/取消 |

**分析图表**：

1. **KPI 指标卡（4个）**
   - 平均损失率：行业参考值 < 5%
   - 平均贫化率：行业参考值 < 8%
   - 平均品位偏差：实际品位与预测品位的差值
   - 异常数据占比：异常记录占总记录的百分比

2. **损失率与贫化率趋势**
   - **Tab1：趋势折线图**
     - 实线：损失率，虚线：贫化率
     - 不同区域用不同颜色区分
     - 红星标记：异常数据点
     - 灰色点线：5%损失率警戒线、8%贫化率警戒线
   - **Tab2：区域对比箱线图**
     - 对比各区域损失率和贫化率的分布差异
     - 直观查看中位数、四分位数、离群值

3. **品位对比分析**
   - **散点图**：实际品位 vs 地质预测品位
     - X轴：预测品位，Y轴：实际品位
     - 灰色虚线：理想线（实际=预测），点越靠近线越准确
     - 正常点：圆点；异常点：红色X标记
   - **统计面板**：品位偏差的详细统计数据

4. **异常数据分析**
   - **饼图**：异常原因分布（地质条件变化、爆破质量不佳、设备故障等）
   - **柱状图**：各区域异常率对比
   - **异常明细表**：可展开查看每条异常记录的详细信息

**使用场景**：
- 地质部门：分析品位预测准确率
- 生产部门：监控损失贫化指标达标情况
- 质量管理：追溯异常数据产生原因

### 3.5 爆破效果

**功能定位**：优化爆破参数，找到最佳炸药单耗区间

**筛选控件**：
| 控件名称 | 功能说明 | 可选值 |
|---------|---------|-------|
| 爆破区域 | 选择分析的作业区域 | 全部区域 / A区 / B区 / C区 / D区 |
| 时间范围 | 选择分析的时间区间 | 任意日期区间 |
| 仅显示最佳区间数据 | 过滤开关 | 勾选/取消 |

**分析图表**：

1. **KPI 指标卡（4个）**
   - 平均炸药单耗（kg/t）：参考区间 0.45-0.55
   - 平均块度合格率（%）
   - 最佳区间占比：参数落在最佳区间的爆破次数百分比
   - 总爆破次数

2. **核心散点图：炸药单耗 vs 块度合格率**
   - 灰色点：普通数据点
   - 绿色点：落在最佳区间（0.45-0.55 kg/t）且合格率≥85%的数据
   - 绿色背景带：最佳参数区间标记
   - 红色虚线：二次拟合趋势线，直观展示倒U形关系
   - 交互功能：悬停查看单次爆破的具体参数

3. **相关性分析仪表盘**
   - Pearson 相关系数仪表图
   - 自动判断相关强度（强/中/弱/无）和方向（正/负）
   - 数据范围、标准差等统计摘要

4. **趋势预测**
   - 周平均块度合格率历史曲线
   - 移动平均线（平滑短期波动）
   - 基于线性回归的未来4周趋势预测（菱形标记）
   - 自动判断趋势方向（上升/下降）

5. **最佳区间分析**
   - **柱状图**：各炸药单耗区间的平均合格率对比
     - 绿色柱：最佳区间（0.45-0.55）
     - 红色柱：其他区间
     - 误差线：合格率的标准差
   - **对比表**：最佳区间与其他区间的指标对比
   - **优化建议**：量化显示采用最佳区间可提升的合格率

6. **各区域对比**
   - 各区域平均块度合格率对比
   - 各区域最佳参数区间占比对比

**使用场景**：
- 爆破工程师：优化爆破设计参数
- 技术部门：建立爆破参数标准
- 成本控制：在保证块度质量前提下节约炸药成本

### 3.6 成本分析

**功能定位**：全面掌握采矿成本构成，识别成本异常并给出优化建议

**筛选控件**：
| 控件名称 | 功能说明 | 可选值 |
|---------|---------|-------|
| 年份选择 | 限定分析的年份 | 全部年份 / 2024年 / 2025年 |
| 分析类型 | 选择分析视角 | 成本构成分析 / 同比环比分析 / 成本趋势分析 |
| 高亮显示异常数据 | 视觉标记开关 | 勾选/取消 |

**分析图表**：

1. **KPI 指标卡（4个）**
   - 平均单位成本（元/吨）：附带环比变化率
   - 同比变化：与去年同期对比的增减百分比
   - 最高成本项：四项成本中金额最高的类别
   - 成本异常次数：异常记录数及占比

2. **分析类型一：成本构成分析**
   - **堆叠柱状图**：各月四项成本（穿孔/爆破/铲装/运输）的构成
     - 红色虚线：标记异常月份
   - **环形饼图**：平均成本构成占比
   - **多折线图**：各成本项及总成本的变化趋势

3. **分析类型二：同比环比分析**
   - **左图：环比变化率柱状图**（与上月比较）
     - 正值：成本上升；负值：成本下降
     - 灰色零线：无变化基准
   - **右图：同比变化率柱状图**（与去年同月比较）
   - **详细数据表**：可展开查看各月的完整同比环比数据

4. **分析类型三：成本趋势分析**
   - **主图**：单位成本柱状图 + 移动平均趋势线
     - 蓝色柱：正常月份；红色柱：异常月份
     - 黄色虚线：移动平均线
     - 绿色点线：平均值参考线
   - **箱线图**：四项成本的分布统计对比
   - **热力图**：各年月单位成本矩阵，颜色越深成本越高

5. **成本异常预警与优化建议**
   - 若无异常：显示成功提示
   - 若有异常：
     - **左侧**：逐条展示异常记录
       - 异常月份
       - 异常成本类型及金额
       - 针对性优化建议
     - **右侧**：
       - 异常类型分布饼图
       - 四项成本的通用优化建议清单

**使用场景**：
- 财务部门：月度/季度成本分析报告
- 生产管理：定位成本上升的具体环节
- 预算编制：基于历史数据制定成本目标

## 六、最佳实践建议

### 6.1 数据分析流程
1. **概览先行**：先在系统概览页查看整体指标，判断是否有异常
2. **定位问题**：进入对应模块深入分析异常指标
3. **多维下钻**：利用时间维度切换和下钻功能定位具体问题时段
4. **对比分析**：结合同比环比、区域对比等功能分析原因
5. **导出分享**：使用图表下载和数据复制功能生成分析报告

### 6.2 不同角色使用建议

**管理者（矿长/生产矿长）**：
- 每日查看：系统概览、产量分析
- 每周查看：设备效率、损失贫化
- 每月查看：成本分析、爆破效果

**技术人员（总工/爆破工程师）**：
- 重点关注：爆破效果模块的参数优化
- 深入研究：损失贫化模块的异常原因分析

**设备管理员**：
- 每日关注：设备效率模块的低效设备列表
- 制定检修计划：结合 TOP 5 低效设备的检修建议

**财务/成本会计**：
- 每月重点：成本分析的三种视图切换使用
- 异常追溯：成本异常预警中的具体建议
